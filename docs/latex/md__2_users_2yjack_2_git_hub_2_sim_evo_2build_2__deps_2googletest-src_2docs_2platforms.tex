The Google\+Test code is officially supported on the following platforms. Operating systems or tools not listed below are community-\/supported. For community-\/supported platforms, patches that do not complicate the code may be considered.

If you notice any problems on your platform, please file an issue on the \href{https://github.com/google/googletest/issues}{\texttt{ Google\+Test Git\+Hub Issue Tracker}}.
